# Airbnb Home React Web App

Bu proje, Airbnb tarzında bir evin rezervasyon ve tanıtım sayfasını içeren bir React web uygulamasını içerir.

## Proje Hakkında

Bu web uygulaması, kullanıcıların bir evi incelemelerini, rezervasyon yapmalarını ve n evin tanıtımlarını  sağlayan temel bir Airbnb benzeri bir platformdur.


## Başlangıç

Projeyi yerel bilgisayarınıza klonlayın:

```bash
git clone git@github.com:erkannebil/Airbnb-Home-React-Web-App.git
cd Airbnb-Home-React-Web-App
Gerekli bağımlılıkları yüklemek için:

bash
Copy code
npm install
Uygulamayı başlatmak için:

bash
Copy code
npm start
Tarayıcıda http://localhost:3000 adresine giderek uygulamayı görüntüleyebilirsiniz.

Kullanım
Uygulama, evleri inceleme, rezervasyon yapma ve ev sahiplerinin evlerini tanıtmak gibi temel özellikleri içerir. Kullanıcı dostu bir arayüzle geliştirilmiştir.
